# 初叶🍂OIDC登录插件

用于网站OIDC登录插件，让支持OIDC登录的程序支持QQ群聊/私聊登录。

**当前版本：v1.1.0**

## 功能特性

- 完整的OIDC服务端实现（支持 discovery、authorize、token、userinfo 端点）
- Web管理后台，支持安全入口配置
- 客户端管理（支持 Client ID / Client Secret 验证）
- 支持QQ群聊验证码接收
- 支持QQ私聊验证码发送
- 验证码自动过期清理
- 美观的验证页面和管理界面
- 个性化主题配置（主题色、图标、Favicon）

## 安装

1. 将插件目录 `astrbot_plugin_chuyeoidc` 放入 AstrBot 的 `data/plugins` 目录
2. 在 AstrBot WebUI 的插件管理处启用该插件
3. 在插件配置中设置相关参数
4. 重启 AstrBot 使插件生效

## 配置

在 AstrBot WebUI 的插件配置页面中，设置以下参数：

### Web管理后台设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| web_username | Web后台用户名 | yeoidc |
| web_password | Web后台密码 | yeoidc |
| web_port | 服务端口 | 33145 |
| secure_path | 安全入口路径 | chuyeoidc |

**⚠️ 安全提示**: 使用默认密码登录时会显示警告，建议立即修改密码！

## 使用方法

### Web管理后台

访问 `http://your-server:33145/{secure_path}` 进入管理后台：

- 默认地址: `http://your-server:33145/chuyeoidc`
- 默认账号: `yeoidc`
- 默认密码: `yeoidc`

管理后台功能：
- **服务信息**: 查看OIDC端点地址和验证状态
- **配置设置**: 个性化设置、验证码设置、验证方式设置
- **认证会话**: 查看当前活跃的认证会话
- **客户端管理**: 管理OIDC客户端（Client ID / Client Secret）

### OIDC端点

插件启动后，提供以下OIDC标准端点：

- **发现文档**: `http://your-server:33145/.well-known/openid-configuration`
- **授权端点**: `http://your-server:33145/authorize`
- **令牌端点**: `http://your-server:33145/token`
- **用户信息**: `http://your-server:33145/userinfo`

### 验证码使用

用户在第三方应用发起OIDC登录后，会显示验证码页面。用户可以通过以下方式验证：

#### 方式一：群聊验证
在配置的群号中发送验证码：
```
验证码 123456
```
或直接发送纯数字验证码：
```
123456
```

#### 方式二：私聊验证
私聊机器人发送验证码：
```
验证码 123456
```
或直接发送纯数字验证码：
```
123456
```

验证成功后，页面会自动跳转或显示"验证成功，点击继续"按钮。

### 第三方应用配置

在支持OIDC的第三方应用中配置：

1. **Issuer**: `http://your-server:33145`
2. **Client ID**: 在管理后台的"客户端管理"中创建的 Client ID
3. **Client Secret**: 对应的 Client Secret
4. **Redirect URI**: 您的应用回调地址

#### 支持的 Scope

- `openid` - 识别账号主体（必需）
- `profile` - 读取昵称、头像等基础资料
- `email` - 读取邮箱标识信息（映射为 QQ 邮箱形式）

## Web后台配置说明

### 个性化设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题色 | 后台页面和验证页面的主题色 | #4f46e5 |
| 图标 URL | 用于后台页面和验证页面的图标 | 默认图标 |
| Favicon URL | 浏览器标签页图标 | 默认图标 |

### 验证码设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 验证码长度 | 验证码的数字位数 | 6位 |
| 有效期 | 验证码有效时间（秒） | 300秒（5分钟） |

### 验证方式设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用群聊验证 | 允许在指定群聊中接收验证码 | 开启 |
| 接收验证码的群号 | 用于接收验证码的QQ群号，多个用英文逗号分隔 | 空 |
| 启用私聊验证 | 允许通过私聊发送验证码 | 开启 |

## 技术说明

### OIDC流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  第三方应用  │────>│  OIDC服务端  │────>│  验证码页面  │
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                                               ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  登录成功   │<────│  Token交换  │<────│  QQ验证码   │
└─────────────┘     └─────────────┘     └─────────────┘
```

1. 第三方应用重定向用户到授权端点
2. 插件生成验证码并显示验证页面
3. 用户在QQ群/私聊发送验证码
4. 验证成功后点击"完成并继续"按钮
5. 第三方应用通过令牌端点获取访问令牌
6. 使用令牌获取用户信息

### 数据存储

- 配置文件存储在 `AstrBot/data/chuyeoidc/` 目录下
- 认证会话存储在内存中
- 验证码默认5分钟过期
- 访问令牌默认1小时过期
- 定期自动清理过期数据

**注意**: 数据存储在 AstrBot 的 data 目录下，更新/重装插件时数据不会丢失。
