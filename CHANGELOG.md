# 更新日志

## 1.0.4 (2025-02-21)
### 安全性改进
- 修复授权码可重放漏洞：授权码在使用后立即失效
- 修复 refresh_token 生命周期逻辑：区分 access_token（1小时）和 refresh_token（30天）的过期时间
- 修复 userinfo 接口 token 类型校验：拒绝使用 refresh_token 获取用户信息
- 修复 CORS 白名单读取错误：支持多 redirect_urls 配置
- 修复审计日志索引维护错误：正确维护操作类型索引
- 修复清空日志实现不完整：同时清空待处理日志和索引
- 修复重定向 URL 拼接逻辑：保留原有查询参数
- 添加模板变量转义防止 XSS 攻击
- 修复 OIDC 元数据签名算法不一致：id_token_signing_alg_values_supported 改为 RS256
